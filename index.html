Castiel SDK Documentation

Install or browse the project on GitHub.

Core Game Functions

Castiel.game(definition)

Registers your game logic. Requires an update(dt) function. Optionally add init() and destroy().

Castiel.game({
  init() { Castiel.log('Game started'); },
  update(dt) { /* logic */ }
});

Castiel.start()

Starts the game loop. Usually called automatically on page load in `run.html`.

Logging & Debugging

Castiel.log(msg)

Logs messages to the browser console and, when running inside the desktop app, to the host editor.

Castiel.log('Hello Castiel');

Castiel.speak(msg)

Internal logging that can forward messages to the host/editor for diagnostics.

File Operations

Castiel.createFile(path, content)

Creates or overwrites a file in the project folder. When used in a browser-only environment this typically throws or is a no-op; in the desktop app it communicates with the host.

Castiel.createFile('data.txt', 'Hello World');

Castiel.readFile(path)

Reads a file from the project folder. Returns a string or empty string if not available.

Canvas & Drawing Helpers

  • Castiel.clear(color='#1e1e1e') — Clears canvas with background color.
  • Castiel.drawRect(x, y, w, h, color='white') — Draws a filled rectangle.
  • Castiel.drawImage(img, x, y, w, h) — Draws an image object.
  • Castiel.loadImage(src, callback) — Loads an image asynchronously and calls the callback with the loaded image.

Input Handling

  • Castiel.keyDown(key) — Returns true if key is pressed.
  • Castiel.mouse() — Returns { x, y, down } with mouse state.

Audio

  • Castiel.loadAudio(src) — Loads an audio file.
  • Castiel.playSound(audio) — Plays the loaded audio.

Sprites & Helpers

  • new Castiel.Sprite(img, x, y, w, h) — Sprite object with draw function.
  • Castiel.rectCollide(a, b) — Detects collision between two rectangles.
